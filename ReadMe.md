## 推荐阅读

学习某一种设计模式时，推荐先找到对应的包名，然后进入App.java，内部包含了每种设计模式的推荐阅读顺序，比如：

```java
/***
 * 策略模式推荐代码阅读顺序：
 * @see Duck
 * @see FlyBehavior
 * @see RubberDuck
 */
```
